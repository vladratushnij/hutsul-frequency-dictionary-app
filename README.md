# Частотний словник гуцульських діалектизмів

Вебзастосунок призначений для автоматизованого опрацювання авторських частотних словників, зіставлення лексем із реєстром гуцульських діалектизмів, формування електронних таблиць результатів і статистичних зведень.

## Функціональні можливості

* завантаження частотних словників;
* використання прикладів із папки `data`;
* очищення та нормалізація лексем;
* зіставлення з реєстром гуцульських діалектизмів;
* формування авторських таблиць;
* формування спільної порівняльної таблиці;
* формування таблиці ручної перевірки;
* розрахунок статистичних показників;
* експорт результатів у Excel і ZIP.

## Структура проєкту

* `app.py` - головний файл застосунку;
* `src/` - модулі обробки даних;
* `assets/` - стилі та іконка застосунку;
* `.streamlit/` - базова конфігурація інтерфейсу;
* `data/` - приклади вхідних файлів;
* `output/` - папка для сформованих результатів;
* `requirements.txt` - список залежностей;
* `run.bat` - запуск застосунку у Windows.

## Встановлення

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
run.bat
```

Або вручну:

```bash
streamlit run app.py
```

## Вхідні дані

У папці `data/` розміщено приклади вхідних файлів:

* частотний словник Марії Матіос;
* частотний словник Тараса Прохаська;
* реєстр гуцульських діалектизмів.

Для частотних словників потрібні колонки:

* `lexeme`;
* `class`;
* `frequency`;
* `semantics`.

Для реєстру гуцулізмів потрібні колонки:

* `dialectism`;
* `meaning`;
* `source`;
* `type`;
* `comment`.

## Результати

Після обробки застосунок формує файли:

* `matios_hutsulisms.xlsx`;
* `prokhasko_hutsulisms.xlsx`;
* `combined_hutsulisms.xlsx`;
* `manual_review.xlsx`;
* `statistics_summary.xlsx`;
* `all_results.zip`.

## Примітка

Файли з результатами створюються після запуску обробки даних у застосунку.
